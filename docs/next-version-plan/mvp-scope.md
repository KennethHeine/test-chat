# MVP Scope

## MVP objective
Prove that the platform can convert a long-term goal into a structured GitHub milestone plan and prepare implementation-ready issues.

## In scope
- web-based chat interface
- create long-term goal record
- define intent, goal, and problem
- generate research checklist
- create milestone plan
- generate detailed issue drafts
- store research, decisions, and planning data
- recommend GitHub branch strategy
- prepare execution ordering for issues
- generate prompts/instructions for GitHub Copilot coding agent

## Out of scope for early MVP
- fully autonomous merge-to-main
- complex multi-repo orchestration
- advanced portfolio management
- custom IDE experience
- replacing GitHub project management completely
- full compliance automation

## Success criteria
- a user can define a product/system goal in chat
- the system generates a research-backed milestone plan
- each milestone gets detailed issue drafts
- the output is good enough to start implementation in GitHub with minimal manual rewriting
